The Kid Should See This

Postado em 2016.06.20 · Tags curadoria, videos

Olá, tudo bem? Gostaria de apresentar para você o site The Kid Should See This (em tradução livre, “a criança deveria ver isto”). A proposta deste site é reunir vídeos interessantes para crianças (e para os adultos também), mas que não sejam necessariamente voltados ao público infantil. Mas por que estou mostrando isso? Bom, veja que história interessante! A diretora de design Rion Nakaya, mãe de duas crianças de 5 e 8 anos, começou o projeto simplesmente reunindo vídeos que ela achava interessantes para mostrar aos filhos, que agora também ajudam a mãe nessa curadoria de materiais.…
